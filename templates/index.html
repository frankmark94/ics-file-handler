About ICS Files

ICS (iCalendar) files are plain text files used to share calendar events, meetings, reminders, and tasks. They use a standardized format defined by RFC 5545.

A simple ICS file structure looks like this:

BEGIN:VCALENDAR
VERSION:2.0
PRODID:-//Example Corp.//CalDAV Client//EN
BEGIN:VEVENT
UID:12345678@example.com
DTSTAMP:20250311T130000Z
DTSTART:20250315T133000Z
DTEND:20250315T150000Z
SUMMARY:Team Meeting
DESCRIPTION:Weekly team sync-up
LOCATION:Conference Room A
END:VEVENT
END:VCALENDAR

This tool helps you create and parse these files without needing to understand the technical details.
